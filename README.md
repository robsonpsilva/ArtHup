# ArtHup | Storytelling-Driven Marketplace

**ArtHup** is a sophisticated digital ecosystem designed to bridge the gap between independent creators and global consumers. By balancing commercial efficiency with artistic identity, the platform empowers high-value niches—from bespoke furniture to organic gastronomy—through a storytelling-driven commerce engine.

---

## 🚀 Technical Stack

*   **Frontend:** React (SPA) with **AAA Usability Compliance**.
*   **Backend:** Node.js + Express.js.
*   **Database:** PostgreSQL (Hybrid Relational-Documental with **JSONB**).
*   **Authentication:** Google OAuth 2.0.
*   **Infrastructure:** Render.com (Global CDN for Frontend, Web Service for Backend).
*   **Version Control:** GitHub with automated CI/CD.
